Civilization 5 AI Biases

This chart is a reference table of computer personality traits in Sid Meier’s Civilization V. It includes information from the Gods & Kings and Brave New World expansions as well as all DLC civs. This project is based on the chart by Raptor-Llama.

AI Biases range between 2 and 12 and influence the computer player’s decisions. Higher values mean the AI is more likely to do something. The values are scraped directly from the game’s XML source code files.

Navigate this website using the arrow keys, scroll wheel, or by clicking and dragging on the data table. Click on leader icons or trait names to sort the data. Click again to return to the default order.
